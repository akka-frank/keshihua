# 日本泡沫经济前后青年失业率的可视化分析

本项目基于日本官方《劳动力调查》数据，通过基础数据可视化方法，分析日本泡沫经济破裂前后青年群体失业率的变化情况，重点关注不同年龄段及性别之间的差异。

---

## 数据来源

- 日本总务省统计局（Statistics Bureau of Japan）
- Labour Force Survey, Japan
- 时间范围：1970s–2000s
- 主要变量：年份、年龄段、性别、失业率

---

## 可视化内容

项目主要使用以下几类基础可视化图表：

- **折线图**：展示不同年龄段失业率的时间变化趋势  
- **分组柱状图**：对比泡沫经济前后各年龄段平均失业率  
- **热力图**：展示时间 × 年龄段的失业率变化结构  
- **箱线图**：分析失业率分布特征及性别差异  

---

## 主要结论

- 泡沫经济破裂后，青年失业率出现明显且持续的上升。
- 15–24 岁群体受冲击最为显著。
- 在青年群体中，失业率在性别维度上存在一定差异。
- 箱线图显示泡沫破裂后失业风险分布明显扩散。
